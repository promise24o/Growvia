# Commission Models

## 💰 Commission Types

### Cost Per Action (CPA)
Fixed amount per conversion:
```
Signup: ₦500
Purchase: ₦2000
```

### Cost Per Sale (CPS)
Percentage of sale:
```
10% of order value
Min: ₦500
Max: ₦5000
```

### Cost Per Lead (CPL)
Fixed amount per qualified lead:
```
Contact Form: ₦300
Quote Request: ₦1000
```

## ⚙️ Configuration

- Payout amount or percentage
- Minimum/maximum payout
- Currency
- Conversion window
- Attribution model
